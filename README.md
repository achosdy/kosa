# Косынка Solitaire PWA

Локальная HTML/PWA-игра «Косынка» для GitHub Pages. Проект не требует серверов, баз данных, CDN или внешних интеграций.

## Что есть

- Полноценная раздача Klondike Solitaire / «Косынка».
- Счетчик ходов, таймер и рекорд отдельно для каждой темы.
- Новая игра, перезапуск текущей раздачи, отмена хода, подсказка.
- Экран окончания: `You won` при победе и `Game over`, если партия заблокирована.
- PWA: manifest, service worker, иконки для iOS, macOS, Windows и Android/Chrome.
- Адаптация под desktop и mobile, включая iPhone Safari.
- Темы: дефолтная, светлая, темная, блочная, Halloween, милая розовая, Чернобыль, подводная.

## Как загрузить на GitHub Pages

1. Скопируй все файлы из архива в корень репозитория.
2. В GitHub открой `Settings` → `Pages`.
3. В разделе `Build and deployment` выбери `Deploy from a branch`.
4. Укажи ветку `main` и папку `/root`.
5. Открой ссылку GitHub Pages после публикации.

## Установка как приложения

- Windows / Chrome / Edge: кнопка «Скачать приложение» или значок установки в адресной строке.
- macOS: установка через браузер с поддержкой PWA.
- iPhone: открыть сайт в Safari → «Поделиться» → «На экран Домой».

## Файлы

- `index.html` — разметка приложения.
- `style.css` — адаптивный дизайн, темы и анимации.
- `app.js` — логика игры.
- `manifest.webmanifest` — настройки PWA.
- `sw.js` — офлайн-кэш.
- `assets/icons/` — иконки приложения.

## Важно

Все изображения и элементы сделаны локально через CSS/SVG-подобную отрисовку и не загружаются из интернета. Это удобно для GitHub Pages и не ломается без внешних сервисов.
